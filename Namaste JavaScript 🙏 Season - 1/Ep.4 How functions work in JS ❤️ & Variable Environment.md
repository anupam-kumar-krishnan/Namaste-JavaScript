## _Function_

- Functions are typically defined using the function keyword, followed by a name, parentheses () (which can contain parameters), and curly braces {} enclosing the function's code block. 

<b>

```js
var x = 1;
a();
b();
console.log(x);

function a() {
  var x = 10;
  console.log(x);
}

function b() {
  var x = 100;
  console.log(x);
}

// Output
// 10
// 100
// 1
```
</b>
